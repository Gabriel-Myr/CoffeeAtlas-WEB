# 淘宝咖啡豆爬虫设计文档

**创建日期**: 2026-02-25

## 1. 项目概述

构建一个淘宝咖啡豆商品数据爬虫系统，支持本地和云端定时运行，自动获取并清洗咖啡豆商品数据。

## 2. 需求总结

| 需求项 | 说明 |
|--------|------|
| 爬取内容 | 淘宝咖啡豆商品数据 |
| 定时更新 | 每天/每周/手动触发（可配置） |
| 数据存储 | JSON/CSV 文件格式 |
| 运行方式 | 本地 + 云端（GitHub Actions） |
| 数据清洗 | 去重、过滤无效数据、标准化格式 |

## 4. 核心组件

### 4.1 爬虫脚本 (`scripts/taobao-coffee-spider/`)

| 文件 | 功能 |
|------|------|
| `spider.py` | 爬虫主程序，发起请求、解析页面 |
| `cleaner.py` | 数据清洗模块，去重、过滤、标准化 |
| `config.py` | 配置文件，关键词、延迟、输出路径等 |
| `main.py` | 入口脚本，整合爬取和清洗流程 |
| `requirements.txt` | Python 依赖 |

### 4.2 数据输出

| 文件 | 说明 |
|------|------|
| `data/coffee-beans.json` | 原始爬取数据 |
| `data/coffee-beans-cleaned.json` | 清洗后数据 |
| `data/coffee-beans.csv` | CSV 格式，便于导入 |

### 4.3 定时任务

| 文件 | 触发方式 |
|------|----------|
| `.github/workflows/spider-daily.yml` | 每天 UTC 0:00 运行 |
| `.github/workflows/spider-weekly.yml` | 每周日 UTC 0:00 运行 |
| 手动触发 | GitHub Actions 手动工作流 |

## 5. 数据清洗功能

1. **去重**: 基于商品 ID 去除重复项
2. **过滤无效数据**:
   - 价格为空或异常
   - 标题为空或包含违禁词
   - 销量为 0
3. **标准化字段**:
   - 价格转换为浮点数
   - 销量提取数字
   - 图片 URL 补全
4. **排序**: 按价格/销量排序

## 6. 字段定义

| 字段 | 类型 | 说明 |
|------|------|------|
| `item_id` | string | 商品 ID |
| `title` | string | 商品标题 |
| `price` | float | 价格（元） |
| `sales` | int | 销量 |
| `shop` | string | 店铺名称 |
| `location` | string | 发货地 |
| `image_url` | string | 图片 URL |
| `detail_url` | string | 商品详情页 URL |
| `crawl_time` | string | 爬取时间 |

## 7. 运行方式

### 本地运行

```bash
# 安装依赖
pip install -r requirements.txt

# 手动运行
python scripts/taobao-coffee-spider/main.py

# 或指定参数
python scripts/taobao-coffee-spider/main.py --keyword "咖啡豆" --pages 3
```

### 云端运行（GitHub Actions）

- **定时触发**: 每天/每周自动运行
- **手动触发**: GitHub Actions 页面手动运行

## 8. 配置项

| 配置项 | 默认值 | 说明 |
|--------|--------|------|
| `keyword` | "咖啡豆" | 搜索关键词 |
| `max_pages` | 5 | 最大爬取页数 |
| `delay_range` | (1, 3) | 请求延迟范围（秒） |
| `max_retry` | 3 | 最大重试次数 |
| `output_dir` | "data" | 输出目录 |
