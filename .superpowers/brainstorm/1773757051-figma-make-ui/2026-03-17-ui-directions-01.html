2. 首页布局对比

布局 1：杂志封面式

先给一块强视觉头图和主标题，再往下进入精选豆单、烘焙商、专题入口。

Hero 标题 + 搜索
本周精选 / 新品策展
烘焙商画册 + 推荐阅读
优点：看起来最精致。缺点：Atlas 的产地导航会退到第二层。

布局 2：Atlas 探索式

首页先放洲 / 国家探索，再接豆子列表和烘焙商入口，强化“从产地进入目录”。

搜索 + 洲别入口
国家气泡 / 路径筛选
结果列表 + 烘焙商入口
优点：最有产品特色。缺点：如果用户只想快速找豆子，会多一步。

布局 3：生活方式逛街式

首页先呈现新品、热卖、收藏回看，再用轻量入口去豆子页和烘焙商页。

搜索 + 今日推荐
新品 / 热卖 / 收藏回看
豆子入口 + 品牌入口
优点：更好逛、更像消费产品。缺点：Atlas 会弱化成辅助亮点。
