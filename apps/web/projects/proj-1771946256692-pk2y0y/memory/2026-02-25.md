## 数据结构建议

基于调研结果，建议数据库 roasters 表包含以下字段：

- id (主键)
- name (名称)
- name_en (英文名)
- country (国家)
- city (城市)
- founded_year (创立年份)
- description (描述)
- specialty (特色)
- website (官网)
- logo_url (Logo URL)
- rating (评分)
- is_domestic (是否国产)
- status (状态)
